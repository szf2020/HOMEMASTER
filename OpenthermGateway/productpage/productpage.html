HomeMaster OpenTherm Gateway

The HomeMaster OpenTherm Gateway is a compact DIN-rail automation controller designed for intelligent integration with OpenTherm-compatible boilers and HVAC systems. It combines OpenTherm communication, isolated digital input, relay outputs, dual 1-Wire temperature interfaces, and ESP32 connectivity for reliable local control and monitoring.

Built around the ESP32-WROOM-32U platform, the module supports multiple power input options (24 V DC, 85-265 V AC, or 120-370 V DC), USB-C programming, and native ESPHome integration with Home Assistant. It is suitable for control-cabinet installations where robust field wiring and local automation are required.

DIN rail • OpenTherm • ESP32 • 24 V DC / AC-DC wide input • Home Assistant

Quick Overview

  • ESP32-based OpenTherm heating gateway for DIN-rail cabinets
  • OpenTherm bus interface (OT+ / OT-) with optocoupler isolation
  • 2 x 1-Wire channels for DS18B20-compatible sensors
  • Isolated digital input and relay outputs for field control
  • USB-C programming + Wi-Fi commissioning with Improv
  • ESPHome pre-installed for fast Home Assistant integration

Typical Applications

  • OpenTherm boiler monitoring and control
  • Zone valve and circulation pump control
  • Room and system temperature monitoring via 1-Wire
  • HVAC cabinet integration with local fallback logic
  • Smart heating integration with Home Assistant
  • Retrofit gateway for existing boiler systems

Home Assistant & ESPHome Integration

The OpenTherm Gateway comes with ESPHome pre-installed and supports direct integration with Home Assistant. Wi-Fi onboarding is available via Improv (USB Serial or BLE), without manual flashing for basic deployment.

Quick Setup Process (Improv Wi-Fi)

  1. Mount & Power - Install on DIN rail and connect power input.
  2. Open Improv - Go to improv-wifi.com in Chrome or Edge.
  3. Connect Device - Use USB-C (serial) or Bluetooth LE.
  4. Enter Wi-Fi - Provide SSID and password, then connect.
  5. Auto Discovery - Device appears in Home Assistant / ESPHome Dashboard.

One-Click Import (ESPHome Dashboard)

After provisioning, click Take Control in ESPHome Dashboard to import the official configuration from GitHub.

Cabling and Wirings

Power Supply Options

The OpenTherm Gateway supports three input options: 24 V DC (V+ / 0V), 85-265 V AC (L / N), or 120-370 V DC (L / N + / -). Use only one power input method at a time.

Relay Output Wiring

Relay output is dry-contact and can be used to switch external actuators (zone valves, pumps, backup heaters) through proper external protection devices.

⚠️ External protection required: Relay outputs MUST be protected by external fuse or circuit breaker. Relay outputs are not internally fused.

Cable Recommendations & Shield Grounding

General Routing Rules

  • Route low-level signal cables (1-Wire/OT) separately from mains, relay outputs, contactors, and power wiring.
  • If crossing power cables is unavoidable, cross at 90 degrees.
  • Keep cable runs as short as practical and avoid long parallel runs with high-current conductors.

Temperature Cable (1-Wire)

  • Recommended: shielded 3-core (+5V / DATA / GND).
  • High-EMI / long runs: shielded pairs + overall shield (e.g., LI2YCY PiMF 2x2x0.50).
  • Topology: daisy-chain (bus), avoid star wiring.
  • Keep sensor stubs <= 0.5 m.
  • DATA pull-up: 4.7 kOhm typical; 2.2-3.3 kOhm for long/heavy loads.

OpenTherm Cable

  • Use twisted pair cable for OT+ / OT-.
  • Overall shield recommended in industrial/noisy cabinets.
  • Examples: J-Y(ST)Y 2x2x0.5 mm2 or LI2YCY PiMF 2x2x0.50.

Shield Grounding

  • Default recommendation: bond cable shield(s) to cabinet PE/EMC ground at controller side only.
  • Do not connect shields directly to signal terminals (1-Wire/OT).
  • If both ends are equipotential bonded cabinets, both-end shield bonding may be used with proper 360-degree clamps.

Safety and Installation Notes

  • Disconnect all power before installation or wiring changes.
  • Use only one power input method at a time (24 V DC or AC/DC L/N input).
  • Install only inside a closed control cabinet; protect all terminals from accidental contact.
  • Use external overcurrent protection (fuse/breaker) for relay circuits and mains inputs.
  • Relay outputs are not internally fused; use external contactors for loads above 3 A and for high inrush/inductive loads.
  • L / N terminals may carry hazardous voltage; installation by qualified personnel only.
  • Follow local electrical code and boiler manufacturer OpenTherm wiring requirements.
